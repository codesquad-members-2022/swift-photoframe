- ### UILabel 클래스 속성(Property)는 어떤게 있는지 학습한다.

  - textAlignment: 텍스트의 정렬기준(default: natural)
    - left: 왼쪽정렬
    - right: 오른쪽 정렬
    - center: 가운데 정렬
    - justified: 행 맞춤 정렬( lineBreak의 설정에 따라 변화가 보임, 행의 넓이에 맞춰 띄어쓰기 간격이 조절 )
    - natural: 지역 기준 정렬( 한국어나 영어는 left, 아랍어와 같은 오른쪽에서 왼쪽으로 읽는 문화권은 right)

  - baselineAdjustment: asdfasfd(default: alignBaselines)
    - 출처: https://goodyoda.tistory.com/392 [GoodYoda's Story]
    - 이 속성은 numberOfLines 값이 1인 경우에만 유효합니다
    - 옵션
      - alignBaselines: 글자의 아래부분(Baseline)이 정해진 위치 기준으로 정렬됨
      - alignCenters: 글자가 View Bound 세로 기준으로 중앙 정렬됨
      - none: 글자가 View Bound 세로 기준으로 좌상단 정렬됨. (기본값)

  - numberOfLines: 보여지는 라인 수(default: 1)
  - lineBreakMode: 화면영역을 넘어가는 텍스트일 경우 처리(default: byTruncatingTail)
    - byWordWrapping: 줄 넘김 기준을 띄어쓰기로 구분
    - byCharWrapping: 줄 넘김 기준을 한글자로 구분
    - byClipping: 영역의 기준으로 잘라서 보여짐
    - byTruncatingTail: 텍스트의 끝부분을 ...처리
    - byTruncatingHead: 텍스트의 앞부분을 ...처리
    - byTruncatingMiddle: 텍스트의 중간을 ...처리

- ### 버튼에 IBAction을 추가할 때 이벤트(Event) 종류에는 어떤 것들이 있는지 학습한다.

  - Did End On Exit : 편집 후 포커스 없어짐

  - Editing changed : 편집

  - Editing Did Begin : 편집할 때

  - Editing Did End : 편집 종료할 때

  - Touch Cancel : 터치 취소할 때

  - Touch Down : 터치 다운시

  - Touch Down Repeat : 여러번 터치 다운시

  - Touch Drag Enter : 드래그하여 객체에 들어갔을 때

  - Touch Drag Exit : 드래그하여 객체 벗어날 때

  - Touch Drag Inside : 객체를 터치하고 드래그 할때

  - Touch Drag Outside : 객체를 터치하고 드래그 하여 객체 밖으로 나올때

  - Touch Up Inside : 컨트롤에서 터치 업 할때

  - Touch Up Outside : 통제가 안되는 영역에서 터치 업할때

  - Value Changed : 값이 변경될 때

    ```
    //그냥 한번 터치 했을때
    touchDown
    touchUpInside
    
    //드래그할때
    touchDown
    touchDragInside
    ....
    touchDragInside
    touchUpInside
    
    //드래그 중 영역을 벗어나고 터치 끝
    touchDown
    touchDragInside
    ...
    touchDragInside
    touchDragExit
    touchDragOutside
    ....
    touchDragOutside
    touchUpOutside
    
    //드래그 중 영역을 벗어나고 다시 영역으로 들어오고 터치 끝
    touchDown
    touchDragInside
    ...
    touchDragInside
    touchDragExit
    touchDragOutside
    ....
    touchDragOutside
    touchDragEnter
    touchDragInside
    touchUpInside
    
    //더블 터치 했을때
    touchDown
    touchUpInside
    touchDown
    touchDownRepeat
    touchDownRepeat
    touchUpInside
    ```

    

  

- ### 버튼에 액션을 여러개 추가할 수 있을까?

  - 아래와 같이 테스트 코드를 작성했다

    ```swift
    override func viewDidLoad() {
      super.viewDidLoad()       
      //addTarget으로 버튼 이벤트 연결하기
      testButton.addTarget(self, action: #selector(onTestButtonTapped_1), for: .touchUpInside)
      testButton.addTarget(self, action: #selector(onTestButtonTapped_2), for: .touchUpInside)
      testButton.addTarget(self, action: #selector(onTestButtonTapped_3), for: .touchUpInside)
      
      //addAction으로 버튼 이벤트 연결하기 1
      testButton.addAction(UIAction(handler: { _ in
                                              self.onTestButtonTapped_9()
                                             }), for: .touchUpInside)
    
      //addAction으로 버튼 이벤트 연결하기 2
      let action = UIAction(title: "testEvent") { _ in
                                                 self.onTestButtonTapped_10()
                                                }
      testButton.addAction(action, for: .touchUpInside)
    }
     
    @objc
    func onTestButtonTapped_1() {
      print("onTestButtonTapped_1")
    }
      
    @objc
    func onTestButtonTapped_2() {
      print("onTestButtonTapped_2")  
    }
    
    @objc
    func onTestButtonTapped_3() {
      print("onTestButtonTapped_3")  
    }
    
    @IBAction func onTestButtonTapped_6(_ sender: Any) {
      print("@IBAction onTestButtonTapped_6")  
    }
    
    @IBAction func onTestButtonTapped_7(_ sender: Any) {
      print("@IBAction onTestButtonTapped_7") 
    }
    
    @IBAction func onTestButtonTapped_8(_ sender: Any) {
      print("@IBAction onTestButtonTapped_8")  
    }
    
    func onTestButtonTapped_9() {
      print("UIAction onTestButtonTapped_9")
    }
    
    func onTestButtonTapped_10() {
      print("UIAction onTestButtonTapped_10")  
    }
    ```

  - 실행결과

    ```
    @IBAction onTestButtonTapped_6
    @IBAction onTestButtonTapped_7
    @IBAction onTestButtonTapped_8
    onTestButtonTapped_1
    onTestButtonTapped_2
    onTestButtonTapped_3
    UIAction onTestButtonTapped_9
    UIAction onTestButtonTapped_10
    ```

  - 내용

    - 위와같이 테스트 코드를 작성하고, 실행 한 결과 버튼에 등록한 이벤트들 모두 호출되는 결과를 볼 수 있다.
    - 버튼에 등록 한 순서대로 호출이 된다.
    - IBAction으로 연결한 버튼이벤트가 먼저 호출되고, 그 후 코드에서 추가한 이벤트가 발생한다.
    
    

- ### 여러 버튼을 동시에 하나의 액션에 연결할 수 있을까?

  - 아래와 같이 코드를 작성

    ```swift
    testButton1.addTarget(self, action: #selector(onTestButtonTapped(_:)), for: .touchUpInside)
    testButton1.tag = 1
    testButton2.addTarget(self, action: #selector(onTestButtonTapped(_:)), for: .touchUpInside)
    testButton2.tag = 2
    testButton3.addTarget(self, action: #selector(onTestButtonTapped(_:)), for: .touchUpInside)
    testButton3.tag = 3
    
    @objc
    func onTestButtonTapped(_ sender: UIButton) {
      print("onTestButtonTapped - \(sender.tag)")
    }
    ```

  - 실행결과

    ```
    onTestButtonTapped - 1
    onTestButtonTapped - 2
    onTestButtonTapped - 3
    ```

  - 내용

    * 위와같이 테스트 코드를 작성하고, 실행한 결과 여러 버튼에 같은 이벤트 함수를 연결할 수 있었다

    * 모두 정상적으로 이벤트 호출이 발생했다

      * 추가로 인자값으로 버튼외에 다른 타입을 받아올 수 있는 방법도 한번 연구해 보았다. 일반적인 방법으로는 안되었다.

        1. 커스텀 버튼 제작

           ```swift
           class CustomButton: UIButton {
               var testValue: String = ""
           }
           ```

        2. 스토리보드에 커스텀버튼으로 연결

           ![스크린샷 2022-02-16 오후 8 53 35](https://user-images.githubusercontent.com/5019378/154259453-87242510-ae16-4e0a-bb77-58cc777b391d.png)

        3. 버튼 이벤트 연결

           ```swift
           testButton1.addTarget(self, action: #selector(onTestButtonTapped(_:)), for: .touchUpInside)
           testButton1.testValue = "testButton1"
           testButton2.addTarget(self, action: #selector(onTestButtonTapped(_:)), for: .touchUpInside)
           testButton2.testValue = "testButton2"
           testButton3.addTarget(self, action: #selector(onTestButtonTapped(_:)), for: .touchUpInside)
           testButton3.testValue = "testButton3"
           ```

        4. 결과 확인

           ```
           onTestButtonTapped - testButton1
           onTestButtonTapped - testButton2
           onTestButtonTapped - testButton3
           ```
